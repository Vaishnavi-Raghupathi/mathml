# mathml

Vite + React + TypeScript scaffold for MathML.

## Stack
- Vite + React + TypeScript
- Tailwind CSS v3
- react-router-dom v6
- react-katex + katex
- d3 v7
- mathjs

## Run locally
1. Install Node.js 18+.
2. Install dependencies.
3. Start development server.

```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Notes
- Tailwind entry styles are in `src/styles/globals.css`.
- Client routing fallback for Vercel is configured in `vercel.json`.
- Module pages are placeholders and are ready for module-by-module content.
